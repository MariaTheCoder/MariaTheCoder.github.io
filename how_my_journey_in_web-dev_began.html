How my journey in web-development began

Posted: October 18th 2020

My first encounter with programming happened at the University of Aarhus in 2014 when I was enrolled in a mandatory 3 months introduction course in object orientated programming. In this course the programming language Java was in focus.

My first experience with learning how to code is a very fond one. Back then I had a lot of fun with learning how to create objects, arrays containing objects, how to filter and sort elements inside of an array given certain criteria and print out the result etc.

It was my very first experience with learning how to code and I found myself spending hours upon hours going through tutorials handed out by tutors, going through previously used examination papers as well as cleaning up my old code.

While I definitely enjoyed this course, it wasn't until 2020 that my attention was once again switched back to programming. My interest in programming was rekindled after I watched a YouTube video on JavaScript done by Mosh Hamedani shared by my boyfriend.

My boyfriend, who has a well-established career in IT, went through the video and explained how the JavaScript code worked. I found myself so intrigued that I soon signed up for the full JS course created by Mosh Hamedani. A paid for course worth every penny!
